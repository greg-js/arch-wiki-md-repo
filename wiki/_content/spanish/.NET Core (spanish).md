**.NET Core** es un nuevo *framework* FOSS de Microsoft para C#, Visual Basic y F#. Está diseñado para ser multiplataforma, modular y apto para aplicaciones modernas, a diferencia de su predecesor, el *framework* .NET.

## Instalación

Si solo desea ejecutar aplicaciones gestionadas por .NET Core, [[install (Español)|instale] el paquete [dotnet-runtime](https://www.archlinux.org/packages/?name=dotnet-runtime).

Para crear aplicaciones con .NET Core, instale también [dotnet-sdk](https://www.archlinux.org/packages/?name=dotnet-sdk).

Microsoft recomienda utilizar [Visual Studio Code](/index.php/Visual_Studio_Code "Visual Studio Code"), su IDE FOSS basado en [Electron](https://electron.atom.io/), para crear y depurar aplicaciones .NET Core.
