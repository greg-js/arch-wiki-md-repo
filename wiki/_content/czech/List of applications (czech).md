## Webový prohlížeč

### Firefox

Věčně populární webový prohlížeč Firefox je dostupný přes pacmana, nicméně nemá své oficiální označení, a tak se program při otevření objevuje pod svým vývojovým kódovým názvem *Gran Paradiso*.

Instalujte s:

```
# pacman -S firefox

```

### Pluginy

Určitě si nainstalujte 'flashplugin', 'mplayer', 'mplayer-plugin', a balíky 'codecs' pro plnohodnotný zážitek z webu:

```
# pacman -S flashplugin mplayer codecs

```

*Nyní je tu i balíček flashplugin pro 64-bity. Pokud flash na x86_64 po spuštění předcházejícího příkazu stále nejde, zkuste níže uvedené rady.*

*   *Poznámka* Povedlo se mi zprovoznit pluginy pouze symlinkováním všeho z '/usr/lib/mozilla/plugins' do '~/.mozilla/plugins', přesněji tedy spuštením příkazu 'mkdir ~/.mozilla/plugins && ln -s /usr/lib/mozilla/plugins/* ~/.mozilla/plugins'.

(Balík codecs obsahuje většinu kodeků, včetně těch pro Win32, QuickTime a RealPlayer9 obsah.)

### Gecko Media Player

Dobrá náhrada v současnosti zastarávajícího *mplayer-plugin* je [Gecko Media Player](http://code.google.com/p/gecko-mediaplayer/). Stabilnější v kombinaci s MPlayer 1.0RC2\. *(Už žádné pády na Apple Trailers.)*

```
#pacman -S gecko-mediaplayer

```

*(**Poznámka!** Odstraňte mplayer-plugin, pokud je již nainstalován.)*

Pro správu vašich e-mailů je užitečný Thunderbird. Pokud používáte GNOME, můžete se spíše podívat na Epiphany a Evolution; pokud používáte KDE, vaší volbou může být zas Konqueror a KMail. Pokud chcete něco zcela jiného, můžete používat Operu. # pacman -S mplayer-plugin

## Hudební přehrávač

### Gnome/Xfce=

#### Exaile

[Exaile](/index.php/Exaile "Exaile") je hudební přehrávač napsaný v Pythonu, který využívá toolkitu GTK+. Pokouší se být blízký oblíbenému Amaroku, nicméně v GTK. Je v repozitáři [community], takže instalujte pomocí:

```
# pacman -S exaile

```

#### Rhythmbox

[Rhythmbox](http://www.gnome.org/projects/rhythmbox/) je integrovaná aplikace pro správu hudby, původně inspirovaná Apple iTunes. Je to svobodný software navržený pro správnou funkčnost pod GNOME desktopem a je založený na multimediálním frameworku GStreamer.

Rhythmbox má mnoho schopností včetně:

*   Hudebního prohlížeče jednoduchého k použití
*   Vyhledávání a řezení
*   Rozsáhlou podporu zvukových formátů skrz GStreamer
*   Podpory internetových rádií
*   Playlistů

Pro instalaci rhythmboxu zadejte:

```
# pacman -S rhythmbox

```

#### Quod Libet

[Quod Libet](http://www.sacredchao.net/quodlibet) je hudební správce, který využívá pro přehrávání zvukových souborů multimediální framework GStreamer. Toto mu umožňuje přehrát všechny typy souborů, které může přehrát Rhythmbox (jenž také používá GStreamer). Quod Libet je vhodnější pro ty, co nepoužívají Gnome desktop, jelikož je menší a má méně závislostí než Rhythmbox (Rhythmbox závisí na Nautilu, jenž vyžaduje mnoho součástí GNOME).

Kromě správce/přehrávače hudby Quod Libet obsahuje editor tagů Ex False.

Mezi schopnosti Quod Libet patří:

*   Hudební prohlížeč jednoduchý k použití
*   Vyhledávání
*   Rozsáhlá podpora zvukových formátů skrz GStreamer
*   Jednoduchá správa playlistů

Pro instalaci Quod Libet zadejte:

```
# pacman -S quodlibet

```

## Vypalování CD a DVD

### GNOME

#### Brasero

[Brasero](http://www.gnome.org/projects/brasero/) je aplikace, která vypaluje CD a DVD pro GNOME desktop. Je navržena tak, aby byla tak jednoduchá, jak je to možné a má specifické schopnosti, které dovolují uživatelům vytvořit své disky rychle a snadno.

Pro instalaci:

```
# pacman -S brasero

```

### KDE

#### K3b

K3b (od **K**DE **B**urn **B**aby **B**urn) je svobodná aplikace pro vytváření CD a DVD navržená pro KDE, určená pro GNU/Linux a ostatní operační systémy podobné UNIXu. Jako v případě většiny KDE aplikací je K3b napsán v programovacím jazyce C++ a používá Qt GUI toolkit. K3b poskytuje grafické uživatelské rozhraní pro provedení většiny úloh při vypalování CD/DVD, jakými mohou být například vytváření Audio CD ze sady zvukových souborů nebo kopírování CD/DVD, stejně jako pokročilejší úlohy jako vypalování eMoviX CD/DVD. Umí také provádět kopie z disku na disk. Program má mnoho výchozích nastavení, která mohou být pokročilejšími uživateli upravena. Vlastní nahrávání na disk je v K3b obstaráno utilitami z příkazového řádku cdrecord nebo wodim, cdrdao a growisofs. Od verze 1.0 nabízí K3b vestavěný DVD ripper – licencovaný pod GPL.

K3b byl většinou (70%) hlasujících na LinuxQuestions.org vyhlášen Multimediální utilitou roku 2006.

Pro instalaci:

```
# pacman -S k3b

```

#### cdrecord, graveman...

Většina vypalovacích utilit pro CD jsou nadstavby nad cdrecord:

```
# pacman -S cdrkit

```

Pokud instalujete balíčky pro vypalování CD/DVD jako Brasero nebo K3B, nainstalují se pro ně také vypalovací knihovny jako libburn nebo cdrkit.

Dobrý nástroj pro vypalování DVD do příkazové řádky je growisofs:

```
# pacman -S dvd+rw-tools

```

## TV karty

Je několik věcí, které je třeba udělat, pokud se chcete dívat pod (Arch) Linuxem na televizi. Nejdůležitější úloha je najít, jaký čip používá váš tuner. Nicméně je jich podporováno celkem mnoho. Pro jistotu se podívejte na databázi hardwaru (viz [tento seznam](http://en.opensuse.org/HCL/TV_Cards), například). Jakmile znáte svůj model, zbývá jen několik kroků.

Ve většině případů budete muset použít bttv ovladače (existují další ovladače, třeba [V4L](http://linux.bytesex.org/v4l2/drivers.html)) spolu s moduly I2C. Jejich konfigurace je ta nejtěžší část. Když budete mít štěstí,

```
# modprobe bttv

```

automaticky zjistí kartu (pro výsledky se podívejte na dmesg). V tom případě potřebujete pouze nainstalovat aplikaci, se kterou lze sledovat televizi. Ale na to se podíváme později. Pokud autodetekce selhala, budete se muset podívat na soubor CARDLIST, který je součástí tarballu [bttv](http://dl.bytesex.org/releases/video4linux/), pro zjištění těch správných parametrů pro svoji kartu. PV951 bez podpory rádia by vyžadovalo tento řádek:

```
# modprobe bttv card=42 radio=0

```

Některé karty potřebují následující řádek, aby vydávaly zvuk:

```
# modprobe tvaudio

```

Nicméně to se různí, takže to prostě vyzkoušejte. Některé jiné karty vyžadují následující řádek:

```
# modprobe tuner

```

Toto také je věc pokusu a omylu.

Abyste konečně mohli sledovat TV, nainstalujte balíček xawtv pomocí:

```
# pacman -S xawtv 

```

a přečtěte si jeho manuálovou stránku.

## Digitální fotoaparáty

Většina novějších digitálních fotoaparátů jsou podporovány jako zařízení USB Mass Storage, což znamená že je můžete prostě zastrčit a zkopírovat si obrázky. Starší fotoaparáty mohou používat PTP (Picture Transfer Protocol), jenž vyžaduje "zvláštní ovladač". gPhoto2 tento ovladač poskytuje a umožňuje přenos obrázků v shellu; digikam (pro KDE) a gthumb (pro GNOME, gtkam by byla další volba) používají tento ovladač a nabízí navíc pěkné grafické uživatelské rozhraní.

## USB klíčenky / pevné disky

USB klíčenky a pevné disky jsou přímo podporovány ovladačem zařízení USB Mass Storage a objeví se jako nové SCSI zařízení (/dev/sdX). Pokud používáte KDE nebo GNOME, měli byste používat též dbus a hal (přidejte je mezi své daemony v /etc/[rc.conf](/index.php/Rc.conf "Rc.conf")), pak budou samočinně připojovány.
