## التثبيت

### الحزم الحالية

**ملاحظة:** تحتاج هذه الحزم إلى إعداد "طلب الوصول إلى ويندوز 7/8 و/أو مايكروسوفت أوفيس 2007" أو وسيط تثبيت، consult corresponding [PKGBUILD](/index.php/PKGBUILD "PKGBUILD") for details.

المتوفّر في [مستودع مستخدِم آرتش](/index.php/Arch_User_Repository_(%D8%A7%D9%84%D8%B9%D8%B1%D8%A8%D9%8A%D8%A9) "Arch User Repository (العربية)"):

*   [ttf-win7-fonts](https://aur.archlinux.org/packages/ttf-win7-fonts/) — خطوط ويندوز 7
*   [ttf-office-2007-fonts](https://aur.archlinux.org/packages/ttf-office-2007-fonts/) — خطوط مايكروسوفت أوفيس 2007
*   [ttf-ms-win8](https://aur.archlinux.org/packages/ttf-ms-win8/) — خطوط ويندوز 8

### الحزم التراثية

**ملاحظة:** الخطوط المُوفّرة بهذه الحزم عتيقة (out-of-date) وينقصها modern hinting instructions وأطقم المحارف الكاملة. يُنصح باستخدام الحزم أعلاه.

[ttf-ms-fonts](https://aur.archlinux.org/packages/ttf-ms-fonts/) متوفّرة في [مستودع مستخدِم آرتش](/index.php/Arch_User_Repository_(%D8%A7%D9%84%D8%B9%D8%B1%D8%A8%D9%8A%D8%A9) "Arch User Repository (العربية)").

وفقَ [اتفاقية رخصة مايكروسفت الأصلية للمستخدم النهائي](http://web.archive.org/web/20020227054122/www.microsoft.com/typography/fontpack/eula.htm)، توجد "بعض" الحدود القانونية عند استخدام هذه الخطوط.

الحزمة تتضمّن:

*   [Andalé Mono](https://en.wikipedia.org/wiki/Andal%C3%A9_Mono "wikipedia:Andalé Mono")

## الخطوط الحادّة تشبه ويندوز XP

لدليل مكتمل حول جعل خطوط مايكروسفت تبدو كما في ويندوز XP موجودٌ في [http://www.sharpfonts.co.cc/](http://www.sharpfonts.co.cc/) (الموقع مُغلق، جرّب [sharpfonts](https://aur.archlinux.org/packages/sharpfonts/) في [مستودع مستخدِم آرتش](/index.php/Arch_User_Repository_(%D8%A7%D9%84%D8%B9%D8%B1%D8%A8%D9%8A%D8%A9) "Arch User Repository (العربية)")). بتفاصيل أقلّ، ثبّت الخطوط كما شُرِح بالأعل واستخدم ملفات XML التي عدّلها الكاتب.

## الخطوط الناعمة تشبه ويندوز 7

استخدم [Infinality's patched freetype2 package](/index.php/Font_configuration#Infinality "Font configuration")، واستخدم profile ويندوز 7 في `local.conf` الموفّر.
