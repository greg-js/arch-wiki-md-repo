# Wii Hacking

A few starting packages for hacking the Wii on Arch Linux. Also see [Wiibrew.org](http://wiibrew.org/) for a guide to getting homebrew setup on the Wii.

## Native Coding

*   [wiiload](https://aur.archlinux.org/packages/wiiload/)<sup><small>AUR</small></sup> lets you upload games to the Homebrew Channel
*   [devkitppc](https://aur.archlinux.org/packages/devkitppc/)<sup><small>AUR</small></sup> for a basic toolchain (compiler) for C and C++ development.
*   [libogc](https://aur.archlinux.org/packages/libogc/)<sup><small>AUR</small></sup> for libraries that work with the Wii, for working with graphics, wiimote, etc.
*   [sdl-wii-svn](https://aur.archlinux.org/packages/sdl-wii-svn/)<sup><small>AUR</small></sup><sup>[[broken link](/index.php/ArchWiki:Requests#Broken_package_links "ArchWiki:Requests"): archived in [aur-mirror](http://pkgbuild.com/git/aur-mirror.git/tree/sdl-wii-svn)]</sup> - a library for SDL on the Wii.
*   [libfat-ogc](https://aur.archlinux.org/packages/libfat-ogc/)<sup><small>AUR</small></sup> - a library for FAT handling (used in USB flash drives).

## Wiimote

*   [cwiid](https://www.archlinux.org/packages/?name=cwiid) - a library in C for working with the Wii Remote.
*   [libwiimote](https://aur.archlinux.org/packages/libwiimote/)<sup><small>AUR</small></sup><sup>[[broken link](/index.php/ArchWiki:Requests#Broken_package_links "ArchWiki:Requests"): archived in [aur-mirror](http://pkgbuild.com/git/aur-mirror.git/tree/libwiimote)]</sup> - a library in C for working with the Wii Remote.
*   [xwii](https://aur.archlinux.org/packages/xwii/)<sup><small>AUR</small></sup><sup>[[broken link](/index.php/ArchWiki:Requests#Broken_package_links "ArchWiki:Requests"): archived in [aur-mirror](http://pkgbuild.com/git/aur-mirror.git/tree/xwii)]</sup> - lets you control your computer from your Wii remote (even has Shell command support).
*   [xwii-hypocorism](https://aur.archlinux.org/packages/xwii-hypocorism/)<sup><small>AUR</small></sup><sup>[[broken link](/index.php/ArchWiki:Requests#Broken_package_links "ArchWiki:Requests"): archived in [aur-mirror](http://pkgbuild.com/git/aur-mirror.git/tree/xwii-hypocorism)]</sup>
