日益增长的社区促使了不同的 IRC 频道的建立，可以通过在 freenode 上使用 `/query alis list *archlinux*` 获得 freenode 上所有包含 "archlinux" 的频道列表.

[#archlinux](irc://irc.freenode.net/archlinux) 是 Arch Linux 主要的支援频道, [#archlinux-offtopic](irc://irc.freenode.net/archlinux-offtopic) 是 Arch Linux 主要的交流频道, [IRC channel](/index.php/IRC_channel "IRC channel") 页面有着两个频道的详细介绍.

## 官方 IRC 频道

**[irc.freenode.net](irc://irc.freenode.net)** 上有几个和 Arch Linux 相关的官方频道. (英文是频道里的主要语言)

| 频道 | 描述 |
| [#archlinux](irc://irc.freenode.net/archlinux) | 主要讨论 |
| [#archlinux64](irc://irc.freenode.net/archlinux64) | x86_64 架构讨论 |
| [#archlinux-aur](irc://irc.freenode.net/archlinux-aur) | [AUR](/index.php/AUR "AUR") 一般讨论 |
| [#archlinux-aurweb](irc://irc.freenode.net/archlinux-aurweb) | [aurweb](https://projects.archlinux.org/aurweb.git/) 开发讨论 |
| [#archlinux-bugs](irc://irc.freenode.net/archlinux-bugs) | Bug 相关的讨论 |
| [#archlinux-offtopic](irc://irc.freenode.net/archlinux-offtopic) | 各类离题讨论. |
| [#archlinux-pacman](irc://irc.freenode.net/archlinux-pacman) | [Pacman](/index.php/Pacman "Pacman") 开发讨论 |
| [#archlinux-projects](irc://irc.freenode.net/archlinux-projects) | Arch Linux 相关计划开发与讨论 (mkinitcpio, abs, dbscripts, devtools, ...) |
| [#archlinux-multilib](irc://irc.freenode.net/archlinux-multilib) | Arch Linux Multilib 讨论和打包交流 |
| [#archlinux-security](irc://irc.freenode.net/archlinux-security) | Arch Linux 软件包安全性讨论. |
