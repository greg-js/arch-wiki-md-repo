[Powerline](https://powerline.readthedocs.io/en/master/index.html) is a statusline plugin for vim, and provides statuslines and prompts for several other applications, including zsh, bash, tmux, IPython, Awesome, i3 and Qtile.

# Installation

[Powerline](https://powerline.readthedocs.io/en/master/index.html) can be installed via 2 ways - via pacman and via python-pip.

## Official repositories

Install [powerline](https://www.archlinux.org/packages/?name=powerline) together with [python-pip](https://www.archlinux.org/packages/?name=python-pip) and [powerline-common](https://www.archlinux.org/packages/?name=powerline-common) packages.

## Using ython-pip

Please refer to [[1]](https://powerline.readthedocs.io/en/master/installation.html) for official installation guide via python-pip.

## Airline-vim alternative

There is currently one known alternative to Powerline - [Vim-airline](https://github.com/vim-airline). It is a part of [vim-plugins](https://www.archlinux.org/groups/x86_64/vim-plugins/) and can be installed separatelly as [vim-airline](https://www.archlinux.org/packages/?name=vim-airline). Optionally, install [vim-airline-themes](https://www.archlinux.org/packages/?name=vim-airline-themes).
