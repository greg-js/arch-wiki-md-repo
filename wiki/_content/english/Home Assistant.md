## Install

[Install](/index.php/Install "Install") the [home-assistant](https://www.archlinux.org/packages/?name=home-assistant) package.

## Configuration

Configuration files are stored at `/var/lib/hass/`. If no configuration exists, a default configuration will be written at startup.

## Usage

To start Home Assistant, [start/enable](/index.php/Start/enable "Start/enable") `home-assistant.service`.

By default, the web interface is available at `[http://localhost:8123](http://localhost:8123)`
