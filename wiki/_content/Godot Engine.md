## Installation

### Stable version

You can either install the [AUR](/index.php/AUR "AUR") package [godot](https://aur.archlinux.org/packages/godot/)<sup><small>AUR</small></sup> or download it from their [downloads page](http://godotengine.org/wp/download/).

### Latest Git version

If you want to install from the AUR, use the package [godot-git](https://aur.archlinux.org/packages/godot-git/)<sup><small>AUR</small></sup>. If you want to compile it manually, have a look at [this page](https://github.com/okamstudio/godot/wiki/compiling_linux) from the official wiki.
