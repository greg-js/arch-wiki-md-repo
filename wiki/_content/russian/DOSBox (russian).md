## Настройка

Никакой начальной конфигурации не требуется, однако официальное руководство DOSBox ссылается на конфигурационный файл `dosbox.conf`.

Изначально этого файла не существует, но Вы можете его создать. Для этого просто запустите `dosbox` без параметров:

```
$ dosbox

```

И затем в командной строке DOS введите:

```
Z:\> config -wc dosbox.conf

```

Конфигурационный файл `dosbox.conf` будет сохранён в текущем каталоге. Также вы можете использовать для каждого приложения свой `dosbox.conf`, для этого просто помещайте его в каталог с приложением.

Все настройки описаны на официальном wiki проекта DOSBox: [http://www.dosbox.com/wiki/Dosbox.conf](http://www.dosbox.com/wiki/Dosbox.conf).

## Использование

Положите вашу любимую игру/программу в каталог, затем запустите `dosbox` с именем этого каталога:

```
$ dosbox ./theoldgame/

```

## Советы

*   Если DOSBox захватил фокус мыши/клавиатуры, нажмите `Ctrl+F10`, чтобы освободить его.
