gold — новый линковщик (пришедший на смену существующему GNU ld), разработанный Яном Тэйлором (Ian Taylor). Интегрирован в [binutils](https://www.archlinux.org/packages/?name=binutils) начиная с 2.19 (но по умолчанию не используется). Призван ускорить сборку программ.

## Использование

Если в переменной окружения $PATH путь /usr/local/bin идёт впереди /usr/bin то для использования линковщика достаточно создать ссылку(от рута):

```
ln -s /usr/bin/ld.gold /usr/local/bin/ld

```

чтобы увидеть что используется именно gold линковщик можно посмотреть на вывод версии утилиты ld(в новой сессии):

```
$ ld -v
GNU gold (GNU Binutils 2.24) 1.11

```

Можно также непосредственно указать необходимую опции компилятору на то каким линковщиком собирать программу, пример для компилятора gccgo(для языка [Go](/index.php/Go_(%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9) "Go (Русский)")) это выглядит так:

```
$ gccgo test.go -fuse-ld=gold -o test

```
