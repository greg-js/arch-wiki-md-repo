## Официальные IRC каналы

Вы можете найти официальные обсуждения, связанные с Arch Linux на следующих Internet Relay Chat (IRC) каналах, расположенных в IRC сети **[irc.freenode.net](irc://irc.freenode.net)** :

| Канал | Описание |
| [#archlinux](irc://irc.freenode.net/archlinux) | Главный канал обсуждения, в основном на английском. |
| [#archlinux64](irc://irc.freenode.net/archlinux64) | Канал обсуждения x86_64, в основном на английском. |
| [#archlinux-aur](irc://irc.freenode.net/archlinux-aur) | Общее обсуждение [AUR](/index.php/AUR_(%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9) "AUR (Русский)"). |
| [#archlinux-aurweb](irc://irc.freenode.net/archlinux-aurweb) | Обсуждение разработки [aurweb](https://projects.archlinux.org/aurweb.git/). |
| [#archlinux-bugs](irc://irc.freenode.net/archlinux-bugs) | Обсуждение ошибок (багов). |
| [#archlinux-offtopic](irc://irc.freenode.net/archlinux-offtopic) | Произвольные обсуждения. |
| [#archlinux-pacman](irc://irc.freenode.net/archlinux-pacman) | Разработка и обсуждение [Pacman](/index.php/Pacman_(%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9) "Pacman (Русский)"). |
| [#archlinux-projects](irc://irc.freenode.net/archlinux-projects) | Обсуждение разработки проектов (mkinitcpio, abs, dbscripts, devtools, ...). |
| [#archlinux-multilib](irc://irc.freenode.net/archlinux-multilib) | Обсуждение проекта Arch Linux Multilib и пакетирования. |
| [#archlinux-security](irc://irc.freenode.net/archlinux-security) | Обсуждение ошибок безопасности в пакетах Arch. |

## Неофициальные IRC каналы

| Канал | Обсуждение |
| [#archlinux-arm](irc://irc.freenode.net/archlinux-arm) | Для тех, кто работает с ARM устройствами (Raspberry Pi и т.д.) |
| [#archlinux-newbie](irc://irc.freenode.net/archlinux-newbie) | Здесь вы можете поучиться, попробовать новые вещи и попросить о помощи, не опасаясь насмешек. |
| [#archlinuxppc](irc://irc.freenode.net/archlinuxppc) | Канал обсуждения PPC, в основном на английском. |
| [#archlinux-women](irc://irc.freenode.net/archlinux-women) | Обсуждение гендерного равенства, в основном на английском. |
| [#archlinux-classroom](irc://irc.freenode.net/archlinux-classroom) | Проект по разработке и проведению занятий для сообщества Arch Linux. |
| [#archaudio](irc://irc.freenode.net/archaudio) | Обсуждение всего связанного с [pro-audio](/index.php/Pro-audio "Pro-audio"), включая [ArchAudio](http://archaudio.org/) (сторонний проект). |
