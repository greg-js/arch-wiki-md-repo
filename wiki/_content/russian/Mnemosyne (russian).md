[Mnemosyne](http://mnemosyne-proj.org) это кроссплатформенное программное обеспечение для обучения и запоминания, использующее [алгоритм интервальных повторений](https://ru.wikipedia.org/wiki/Интервальные_повторения)

Mnemosyne вдохновлена проприетарной [SuperMemo](http://www.supermemo.com) и аналогична [Anki](/index.php/Anki "Anki"), но с большим фокусом на минималистичный, не отвлекающий интерфейс, простую и гибкую рабочюю среду.

Mnemosyne написана на Python 2 и использует графический фреймворк [Qt](/index.php/Qt "Qt").

## Установка

### Установка из AUR

Неофициальные пакеты доступны в [пользовательском репозитории Arch](/index.php/Arch_User_Repository_(%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9) "Arch User Repository (Русский)"):

*   [mnemosyne](https://aur.archlinux.org/packages/mnemosyne/) (последняя стабильная версия)
*   [mnemosyne-bzr](https://aur.archlinux.org/packages/mnemosyne-bzr/) (последняя версия для разработчиков из ветки trunk)

### Установка вручную

Скачайте архив с [официальной страницы загрузки](http://www.mnemosyne-proj.org/download-mnemosyne.php) и следуйте инструкциям во вложенном файле README.

### Настройка

Большая часть опций настройки доступна прямо в графическом интерфейсе. Несколько редко используемых опций можно отредактировать в конфигурационном файле `~/.config/mnemosyne/config.py`.

##### Разрешение математических формул

Если вы хотите уменьшить разрешение отображаемых математических формул (по умолчанию - 200 dpi, что излишне на большинстве экранов) для лучшего сочетания с остальным текстом, откройте файл `~/.config/mnemosyne/config.py` и уменьшите число в опции -D в строке вида:

```
dvipng = "dvipng -D 200 -T tight tmp.dvi"

```
