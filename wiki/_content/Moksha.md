**Moksha** is a desktop environment created in 2014\. It is a fork of [Enlightenment](/index.php/Enlightenment "Enlightenment") and it is currently used as default desktop environment in Ubuntu-based Bodhi Linux.

## Installation

You can install Moksha using the [AUR](/index.php/AUR "AUR") package [moksha](https://aur.archlinux.org/packages/moksha/)<sup><small>AUR</small></sup>.

## Other applications

For additional functionality, you may wish to install the following:

*   **moksha-module-deskshow** — Deskshow Moksha module

[https://github.com/JeffHoogland/moksha-modules-extra](https://github.com/JeffHoogland/moksha-modules-extra) || [moksha-module-deskshow-git](https://aur.archlinux.org/packages/moksha-module-deskshow-git/)<sup><small>AUR</small></sup>
