# Category:Boot loaders

In order to boot Arch Linux, a Linux-capable boot loader such as [GRUB](/index.php/GRUB "GRUB"), [Syslinux](/index.php/Syslinux "Syslinux"), [LILO](/index.php/LILO "LILO") or [GRUB Legacy](/index.php/GRUB_Legacy "GRUB Legacy") must be installed to the [Master Boot Record](/index.php/Master_Boot_Record "Master Boot Record") or the [GUID Partition Table](/index.php/GUID_Partition_Table "GUID Partition Table"). The boot loader is responsible for loading the kernel and [initial ramdisk](/index.php/Mkinitcpio "Mkinitcpio") before initiating the [boot process](/index.php/Arch_boot_process "Arch boot process").
